# App Store Screenshots Guide

## Required Sizes

### iPhone (6.9")
- 1320 x 2868 pixels (portrait)
- Show: Gateway connection, Voice Wake, Chat interface

### iPhone (6.7")  
- 1290 x 2796 pixels (portrait)
- Same content as 6.9"

### iPhone (6.5")
- 1242 x 2688 pixels (portrait)
- Legacy size, same content

### iPad Pro (12.9")
- 2048 x 2732 pixels (portrait)
- Show: Canvas rendering, Settings, Multi-tasking

## Screenshot Content

1. **Welcome/Onboarding** - First-time setup with gateway discovery
2. **Gateway Connection** - Active connection with status indicator
3. **Voice Wake** - Voice activation demo with waveform
4. **Chat Interface** - Clean messaging experience
5. **Canvas Mode** - Rich visual content rendering
6. **Settings** - Privacy controls and customization

## Design Guidelines

- Use actual app UI (no mockups)
- Dark mode preferred (matches app)
- Show real features in action
- Include minimal text overlays explaining key features
- Maintain Apple HIG aesthetic
- Ensure all text is legible at thumbnail size
